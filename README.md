# Tech Scout Radar Challenge

A simple browser-based classroom game for **Technology Intelligence and Scouting**.  
This activity is designed for UP MTM / TM201 students to practice evaluating technology signals and deciding whether a technology should be placed under:

- Adopt
- Trial
- Assess
- Hold

The game helps students apply Technology Management concepts such as technology scouting, technology intelligence, technology assessment, strategic fit, risk evaluation, and business impact analysis.

---

## Features

- Single-page HTML, CSS, and JavaScript game
- No installation required
- Works directly in the browser
- Team-based gameplay
- Timer-based challenge
- Scoring system
- Local leaderboard
- Classroom-friendly activity for 10–15 minutes
